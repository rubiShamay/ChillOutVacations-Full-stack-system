הRESR API שלנו בנוי ככה:

בקשות GET :
1- GetAllVactions - קבלת כלל החופשות (משתמשים)
2- GetOneVaction - קבלת חופשה אחת (עדיין לא ידוע אם נשתמש)
3- GetAllFollowers - קבלת כלל המשתמשים ומספר והחופשות אחרים הם עוקבים - פרוצדורה (רק אדמין)

בקשות POST :
1- addVaction - הוספת חופשה (רק אדמין)
2- addFollow - הוספת עוקב לחופשה - פרוצדורה (משתמשים)

בקשות PUT :
1- updateVaction - עדכון חופשה (רק אדמין)

בקשות DELETE :
1- deleteVacatio - מחיקת חופשה (רק לאדמין)
2- deleteFollow - מחיקת עוקב (משתמשים)
